A dynamic **Power BI dashboard** integrating **live Weather API data** and **custom AQI visualizations** to deliver real-time weather insights, pollutant monitoring, and forecast analytics.

## Project Overview
This project connects Power BI to a Weather API to pull **live data** such as temperature, humidity, wind, pressure, and AQI. The dashboard refreshes automatically and provides historical trends, forecasts, and pollutant-level analysis.

## Key Features
- Live API weather integration  
- Custom AQI gauges  
- 7-day forecasting  
- Historical trend analysis  
- City-level weather mapping  

## Tech Stack

- Weather API  
- Power Query (M)
